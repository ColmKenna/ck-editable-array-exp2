ck-editable-array

A powerful, accessible, and flexible custom element for managing array data.

Getting Started

Explore the core features through these foundational examples:

  • Basic CRUD - Add, edit, delete, and restore items
  • Validation - Schema-based validation with field and row-level feedback
  • Read-only Mode - Component in viewing mode with editing disabled

Advanced Features

Dive deeper into powerful capabilities and customization options:

  • Modal Edit Mode - Edit items in a modal dialog instead of inline
  • Advanced Validation - Custom rules, cross-field validation, and conditional requirements
  • Event Handling - Monitor all component events with detailed logs
  • Large Dataset Performance - Handling 100+ rows efficiently with performance metrics

Installation

Import the component module in your project:

import '@colmkenna/ck-webcomponents';

For more details, see the project README.
