Razor (ASP.NET) Example

Using ck-editable-array in ASP.NET MVC / Razor Pages

Note: This page demonstrates how to integrate the ck-editable-array component in ASP.NET MVC or Razor Pages applications. The code examples show typical integration patterns.

Installation

Step 1: Install via npm

npm install @colmkenna/ck-webcomponents

Step 2: Copy to wwwroot

Copy the built files to your wwwroot folder:

wwwroot/
└── lib/
    └── ck-webcomponents/
        ├── ck-editable-array.esm.js
        └── ck-editable-array.min.js

Usage in _Layout.cshtml

Add the component script to your layout file:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>@ViewData["Title"] - MyApp</title>
    <link rel="stylesheet" href="~/css/site.css" />
</head>
<body>
    @RenderBody()

    <script src="~/lib/jquery/dist/jquery.min.js"></script>
    <script src="~/lib/bootstrap/dist/js/bootstrap.bundle.min.js"></script>

    @* Include the web component *@
    <script type="module" src="~/lib/ck-webcomponents/ck-editable-array.esm.js"></script>

    @await RenderSectionAsync("Scripts", required: false)
</body>
</html>

@section Scripts {
    <script>
        document.addEventListener('DOMContentLoaded', function() {
            const list = document.getElementById('userList');
            const addBtn = document.getElementById('addUserBtn');

            // Initialize with data from Model
            list.data = @Html.Raw(Json.Serialize(Model));

            // Configure new item factory
            list.newItemFactory = function() {
                return {
                    id: 0,
                    name: '',
                    email: ''
                };
            };

            // Handle add button
            addBtn.addEventListener('click', function() {
                list.addRow();
            });

            // Listen for data changes and save to server
            list.addEventListener('datachanged', function(e) {
                console.log('Data changed:', e.detail.data);

                // Send updated data to server via AJAX
                fetch('@Url.Action("UpdateUsers", "Home")', {
                    method: 'POST',
                    headers: {
                        'Content-Type': 'application/json',
                        'RequestVerificationToken': '@GetAntiCsrfToken()'
                    },
                    body: JSON.stringify(e.detail.data)
                })
                .then(response => response.json())
                .then(data => {
                    if (data.success) {
                        console.log('Data saved successfully');
                    }
                })
                .catch(error => {
                    console.error('Error saving data:', error);
                    alert('Failed to save changes');
                });
            });
        });
    </script>
}

@functions {
    // Helper to get anti-forgery token
    public string GetAntiCsrfToken()
    {
        return Antiforgery.GetTokens(Context).RequestToken;
    }
}

Example Controller

using Microsoft.AspNetCore.Mvc;
using System.Collections.Generic;
using System.Linq;
using MyApp.Models;

namespace MyApp.Controllers
{
    public class HomeController : Controller
    {
        // Sample in-memory data store
        private static List<User> _users = new List<User>
        {
            new User { Id = 1, Name = "Alice Johnson", Email = "alice@example.com" },
            new User { Id = 2, Name = "Bob Smith", Email = "bob@example.com" }
        };

        public IActionResult Users()
        {
            return View(_users);
        }

        [HttpPost]
        [ValidateAntiForgeryToken]
        public IActionResult UpdateUsers([FromBody] List<User> users)
        {
            // Filter out deleted items
            var activeUsers = users.Where(u => u.Deleted != true).ToList();

            // Update data store
            _users = activeUsers;

            return Json(new { success = true });
        }
    }

    public class User
    {
        public int Id { get; set; }
        public string Name { get; set; }
        public string Email { get; set; }
        public bool? Deleted { get; set; }
    }
}

Best Practices

  1. CSRF Protection: Always include anti-forgery tokens when making POST requests
  2. Data Serialization: Use @Html.Raw(Json.Serialize(Model)) to safely pass model data to JavaScript
  3. Error Handling: Implement proper error handling for AJAX requests
  4. Validation: Implement both client-side (using validationSchema) and server-side validation
  5. CSS Integration: Include stylesheet references in templates if using Bootstrap or custom styles

Form Submission

If you need to submit the data as part of a traditional form:

<form asp-action="SaveUsers" method="post">
    <input type="hidden" id="usersJson" name="usersJson" />

    <ck-editable-array id="userList">
        <!-- templates here -->
    </ck-editable-array>

    <button type="submit" class="btn btn-primary">Save All</button>
</form>

<script>
    document.querySelector('form').addEventListener('submit', function(e) {
        const list = document.getElementById('userList');
        document.getElementById('usersJson').value = JSON.stringify(list.data);
    });
</script>

Common Issues

Self-Closing Tags

Razor doesn't support self-closing custom elements. Always use explicit closing tags: 

<!-- ✗ Wrong -->
<ck-editable-array />

<!-- ✓ Correct -->
<ck-editable-array></ck-editable-array>

HTML Encoding

When passing data from Model to JavaScript, use @Html.Raw(Json.Serialize()) to prevent double-encoding issues.
